# BBS - PHP 掲示板アプリケーション

---

## 概要

BBS は、PHP を使用して開発されたシンプルな掲示板アプリケーションです。ユーザーは投稿やコメントを作成し、管理することができます。

---

## 主な機能

- ユーザー認証（ログイン・ログアウト）
- 投稿の作成・編集・削除
- コメント機能
- ページネーション対応
- セキュリティ対策（CSRF, SQLインジェクション防止）

---

## 使用技術

- **バックエンド**: PHP, MySQL
- **フロントエンド**: HTML, CSS, JavaScript
- **サーバー**: Apache
